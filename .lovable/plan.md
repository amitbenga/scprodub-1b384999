

# שינויים בטופס ההרשמה

## סיכום השינויים

### 1. שירה - פישוט סגנונות
- **רמת שירה** - נשארת כמו שהיא (תפריט בחירה)
- **סגנונות שירה** - משתנים מ"בחירה + רמה לכל סגנון" ל**בחירה מרובה פשוטה** (chips כמו כישורים/שפות). ללא תפריט רמה לכל סגנון
- **סגנונות שירה נוספים** - נשאר כשדה טקסט חופשי אבל **ללא תפריט רמה**. פשוט input עם כפתור הוספה

### 2. דוגמת שירה - הקלטה/העלאה בסקשן מדיה
- הוספת שדה חדש `AudioInput` בסקשן מדיה עבור "דוגמת שירה"
- אותו עיצוב כמו דוגמת הקול הקיימת (הקלטה או העלאה)
- ה-URL יישמר רק ב-`raw_payload` (לא עמודה נפרדת)

### 3. כישורים - הוספת מבטאים
הרשימה הנוכחית: קריינות, מבטא רוסי, כל מבטא אפשרי, אחר
הרשימה החדשה: קריינות, מבטא רוסי, **מבטא צרפתי, מבטא איטלקי, מבטא ספרדי, מבטא גרמני, מבטא אנגלי**, כל מבטא אפשרי, אחר

## פרטים טכניים

### קובץ: `src/components/intake/ActorIntakeForm.tsx`

**שינוי 1 - SKILLS:**
```typescript
// לפני
const SKILLS = ["קריינות", "מבטא רוסי", "כל מבטא אפשרי", "אחר"];

// אחרי
const SKILLS = [
  "קריינות", "מבטא רוסי", "מבטא צרפתי", "מבטא איטלקי",
  "מבטא ספרדי", "מבטא גרמני", "מבטא אנגלי",
  "כל מבטא אפשרי", "אחר"
];
```

**שינוי 2 - State:**
- הסרת `pendingStyle`, `pendingStyleLevel` (לא צריך רמה לכל סגנון)
- הסרת `newOtherStyleLevel` (לא צריך רמה לסגנונות נוספים)
- שינוי `singingStyles` מ-`SingingStyleWithLevel[]` ל-`string[]` (בחירה מרובה פשוטה)
- שינוי `singingStylesOther` מ-`OtherSingingStyle[]` ל-`string[]` (רק שמות)
- הוספת `singingSampleFile: File | null` לדוגמת שירה

**שינוי 3 - סקשן שירה (UI):**
- סגנונות שירה: `ChipSelect` עם בחירה מרובה (כמו כישורים/שפות)
- סגנונות נוספים: input טקסט + כפתור הוספה, ללא select של רמה

**שינוי 4 - סקשן מדיה:**
- הוספת `AudioInput` נוסף עבור "דוגמת שירה"

**שינוי 5 - handleSubmit:**
- העלאת `singingSampleFile` ל-storage (תיקיית audio)
- שמירת ה-URL ב-`raw_payload.singing_sample_url` בלבד
- עדכון מבנה `singing_styles` ו-`singing_styles_other` ב-raw_payload
