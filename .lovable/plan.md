
# תוכנית: הוספת אפשרות הקלטה לדוגמת הקול

## מה נבנה
נוסיף כפתור הקלטה לצד אפשרות ההעלאה הקיימת, שיאפשר למשתמשים להקליט דוגמת קול ישירות מהדפדפן.

## תמיכה בדפדפנים
- **Chrome/Edge/Firefox/Safari (מודרניים)** - נתמכים
- **iOS Safari** - נתמך מגרסה 14.3+
- אם הדפדפן לא תומך או שהמשתמש לא מאשר מיקרופון - נציג הודעה ונשאיר רק את אפשרות ההעלאה

## פרטים טכניים

### קבצים שייווצרו/יעודכנו

#### 1. Hook חדש: `src/hooks/use-audio-recorder.ts`
Hook מותאם אישית לניהול ההקלטה עם MediaRecorder API:
- `startRecording()` - מבקש הרשאת מיקרופון ומתחיל הקלטה
- `stopRecording()` - עוצר ומחזיר Blob
- `pauseRecording()` / `resumeRecording()` - עצירה זמנית
- `clearRecording()` - מאפס
- States: `isRecording`, `isPaused`, `duration`, `audioBlob`, `audioUrl`, `error`, `isSupported`

#### 2. רכיב חדש: `src/components/intake/AudioInput.tsx`
רכיב שמאחד העלאה והקלטה:
- מציג שני כפתורים: "להקלטה" ו"להעלאת קובץ"
- מנהל מצב בין שתי האפשרויות
- מציג ממשק הקלטה עם טיימר ואנימציה
- מאפשר השמעה לפני שליחה
- מחזיר `File` object (גם מהקלטה וגם מהעלאה)

#### 3. עדכון `src/components/intake/ActorIntakeForm.tsx`
- החלפת `<FileUpload type="audio">` ב-`<AudioInput>`
- ללא שינוי בלוגיקת השמירה - ממשיכים לקבל `File` object

#### 4. עדכון `src/lib/file-upload.ts`
- הוספת `audio/webm` לרשימת הפורמטים הנתמכים (פורמט ברירת המחדל של MediaRecorder)

#### 5. עדכון `src/index.css`
- סגנונות לממשק ההקלטה
- אנימציית "pulse" לאינדיקטור הקלטה פעילה

### המרת הקלטה לקובץ
הקלטת MediaRecorder מחזירה Blob. נמיר אותו ל-File object:
```typescript
const file = new File([blob], "recording.webm", { type: "audio/webm" });
```

### מגבלות
- הקלטה מוגבלת ל-2 דקות (אפשר להגדיל)
- פורמט: WebM (נתמך בכל הדפדפנים המודרניים, כולל iOS 14.3+)
